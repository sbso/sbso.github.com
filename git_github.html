github

Git是一个开源的分布式版本控制系统，用以有效、高速的处理各种规模的项目版本管理， 它是 Linux Torvalds 为了帮助管理 Linux 内核开发而开发的一个开放源码的版本控制软件，后来得到广泛的使用。

GitHub可以托管各种git库，并提供一个web界面，但与其它像 SourceForge或GoogleCode这样的服务不同，GitHub的独特卖点在于从另外一个项目进行分支的简易性。为一个项目贡献代码非常简单：首先点击项目站点的“fork”的按钮，然后将代码检出并将修改加入到刚才分出的代码库中，最後通过内建的“pull request”机制向项目负责人申请代码合并。已经有人将GitHub称为代码玩家的MySpace。

由于git大家都应该很熟悉，而hub呢，其实就是基于git的一个开源平台而已，这一点做得很好，如此之简单，为什么当时没有人想到呢？呵呵……我这里也不用多说了。进github 主页注册一个帐号你就知道是怎么操作了。很简单的操作。

简要步骤

  1. install
    安装就不用多讲了。
  2. git初始化设置
    git config --global user.name "Your Real Name"
    git config --global user.email you@email.address
  3. 建立本地仓库
    mkdir lijieliwei lijieliwei(仓库名称)
    git init
    git add . 注意这里的. 是添加当前上当下所有文件的意思。如果你不希望如此可以一个一个地增加文件。比如git add README.txt
    git commit -m 'first commit' 这里的意思是说添加注释的意思。
  1. 注册github帐号
    这也没有什么讲的。
  2. 创建ssh公钥并增加到github
    打开git bash。
    ssh-keygen -t rsa -C 'your@email.address'
    如果不使用密码则直接回车即可。
    ssh -T git@github.com test the github connect status.
    cat ~/.ssh/id_rsa.pub 然后增加此公钥到github.
  3. 上传代码
    git remote add origin git@github.com:你的github用户名/你的github项目名.git
    git push -u origin master
